# Week 2: Purple blob at Sushi Restaurant

## Setup and Interaction Instructions

The little purple blob is going across the sushi restaurnt, it can jump onto blocks to reach the highest one.

**Controls:**

- Move left/right: Arrow Keys or A/D
- Jump: Up Arrow or W
- Fall off the screen to reset to the start
